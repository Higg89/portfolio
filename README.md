# Портфолио — Шуманов Никита Алексеевич

Личный сайт-портфолио фронтенд-разработчика. Адаптивная вёрстка: HTML5, CSS (Grid, Flexbox), доступность (a11y), mobile-first.

## Структура проекта

```
lab8/
├── index.html          # Главная страница
├── css/
│   └── styles.css      # Стили
├── img/                # Изображения
└── README.md
```

## Технологии

- Семантическая разметка (header, main, section, footer)
- CSS Grid для макета страницы
- Flexbox внутри компонентов
- Адаптивность: mobile-first, медиазапросы от 768px
- Галерея: `grid-template-columns: repeat(auto-fit, minmax(280px, 1fr))`
- Доступность: skip-link, контраст ≥4.5:1, фокус, label для полей формы
- Normalize.css для кроссбраузерности
